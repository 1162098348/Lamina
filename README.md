# Lamina
## **1. 概述**
**Lamina** 是一种面向过程的数学计算语言，专注于数学问题的高效解决，支持图灵完备性。其设计目标是提供简洁、直观的语法，同时支持复杂数学运算（如向量、矩阵、递归、循环等）。

---

## **3. 数据类型**
### **3.1 基本类型**
| **类型**       | **描述**                                  | **示例**               |
|----------------|------------------------------------------|-----------------------|
| `int`          | 整数                                    | `42`, `-10`           |
| `float`        | 浮点数                                  | `3.14`, `-0.5`        |
| `bool`         | 布尔值                                  | `true`, `false`       |
| `string`       | 字符串（双引号）                         | `"Hello"`, `"world"`  |
| `null`         | 空值                                    | `null`                |
| `bigint`       | 大整数（需显式声明）                      | `bigint x = 999999;`  |

**注意**：
- `bigint` 类型需要用户显式声明，例如：`bigint result = 25!;`
- 不支持复数类型和单引号字符串

### **3.2 复合类型**
#### **数组（Array）**
- **语法**：`[元素1, 元素2, ...]`
- **示例**：  ```Lamina
  var arr = [1, 2, 3];          // 数组初始化
  var nested = [[1,2], [3,4]];  // 多维数组
  ```

#### **向量（Vector）**
- **语法**：`[元素1, 元素2, ...]`（隐式向量类型）
- **特性**：
  - 支持向量运算（加减、点积、叉积等）。
  - 维度一致性检查（如加法需维度相同）。

#### **矩阵（Matrix）**
- **语法**：`[[行1元素], [行2元素], ...]`
- **示例**：
  ```Lamina
  var m = [[1,2], [3,4]];  // 2x2矩阵
  ```

#### **函数（Func）**
- **定义**：函数可以定义和调用，支持递归。
- **示例**：
  ```Lamina
  func add(a, b) { return a + b; }
  ```

## **4. 配置定义**
### **4.1 递归深度限制**
用户可以通过 `define` 语句设置递归深度限制：
```Lamina
define MAX_RECURSION_DEPTH 200;  // 设置最大递归深度为200
```
- 默认递归深度限制为100层
- 可设置范围：1-10000

### **4.2 大整数类型**
用户需要显式声明 `bigint` 类型变量来处理大整数，隐式转换不再支持：
```Lamina
bigint result = 25!;   // 必须声明 bigint，才能进行大数阶乘
bigint big_num = 999999999999;   // 声明大整数
```

## **13. 示例代码**
## **13. 示例代码**
### **13.1 解二次方程**
```Lamina
func quadratic(a, b, c) {
    var discriminant = b^2 - 4*a*c;
    if (discriminant < 0) {
        print("Complex roots not supported in current version");
        return null;
    } else {
        var root1 = (-b + sqrt(discriminant))/(2*a);
        var root2 = (-b - sqrt(discriminant))/(2*a);
        return [root1, root2];
    }
}

var roots = quadratic(1, -3, 2);
print("Roots:", roots);  // 输出 [2, 1]
```

### **13.2 大整数阶乘计算**
```Lamina
define MAX_RECURSION_DEPTH 30;  // 设置递归深度

bigint result = 25!;             // 计算25的阶乘
print("25! =", result);

// 递归版本
func factorial_big(n) {
    if (n <= 1) return 1;
    return n * factorial_big(n - 1);
}

bigint fact10 = factorial_big(10);
print("10! =", fact10);
```

## **14. 设计原则**
## **14. 设计原则**
1. **简洁性**：语法简洁，符合数学表达习惯。
2. **数学友好**：直接支持向量、矩阵和常用数学运算。
3. **面向过程**：通过函数组织代码，避免复杂对象模型。
4. **图灵完备**：支持无限循环、递归和任意计算。
5. **用户可配置**：递归深度限制、大整数类型等需用户显式设置。
